# Clap Action
## _GitHub Actions Workshop_
## Features

- Basic python project setup
- Code build, test, deploy
- Docs using Mkdocs

## Setup

Clap Action requires Python to run.

```sh
poetry new clap-action
```
## Install

```sh
cd clap-action 
poetry install
```


## Test

```sh
cd clap-action 
poetry run pytest
```
